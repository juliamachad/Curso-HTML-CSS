Estilizações CSS

Vimos algumas estilizações com CSS ao longo dos estudos, mas uma quantidade muito pequena se comparada à tudo que essa linguagem nos permite

Crie mais uma pasta de exercício e um index.html para ir testando o que vou apresentar

Pra começar utilizaremos uma tag nova em html: <div></div>

Ela representa um container genérico, que de certa forma não representa nada. É mais utilizada para agrupar elementos e aplicar estilos. Deve ser utilizado somente quando não houver uma tag específica para o agrupamento do conteúdo.

Aqui utilizaremos apenas para testar os estilos mesmo. Utilizarei esse código:

<div class="container">
    <h1>Testando estilos</h1>
    <p>Este container foi destinado para o teste de estilos em CSS</p>
</div>

Dessa vez faremos as estilizações com um CSS externo. Então crie um arquivo style.css e o link na head do seu html assim:

<link rel="stylesheet" href="style.css">

Para começar, vamos adicionar uma cor ao fundo dessa div para facilitar a visualização das estilizações seguintes.

Lembra como funciona o seletor de classe?

.container{
	 background-color: rgb(146, 166, 204);
}
