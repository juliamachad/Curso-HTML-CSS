Diferença entre HTML, CSS e JS

HTML

Conteúdo

HTML não é um linguagem de programação, mas sim de marcação, ou seja, não pode ser usado para criar funcionalidades dinâmicas. O Hypertext Markup Language, significado da sigla, na verdade serve para organização e formatação de conteúdo

Em qualquer site, todo o texto, as imagens, os vídeos e o conteúdo no geral foram criados com HTML . Observe esse exemplo do site da iJunior, onde este conteúdo (criado com HTML) está destacado:

São as outras linguagens, principalmente o CSS, que o deixará mais apresentável. Veremos isso a seguir.

CSS

Design/Estilo

CSS (Cascading Style Sheets ou Folhas de Estilo em Cascata) é uma linguagem de estilização, usada para modificar a apresentação do conteúdo, descrevendo como os elementos, criados com HTML, serão mostrados na tela.

Preste atenção em todas as propriedades que mudam o visual de um site: as cores, os tamanhos, os espaçamentos, a formatação de textos, a posição dos componentes e organização visual do conteúdo em blocos, linhas e colunas. Tudo isso (e mais um pouco) foi definido com CSS.

Essa linguagem quase mágica que foi responsável por deixar o site da iJunior, que vimos anteriormente, mais bonito. Legal, né?!

JS

Interações

JavaScript (frequentemente abreviado como JS) já é uma linguagem de programação.

No Front de um site, ela é responsável pelas interações dinãmicas. Toda vez que uma página da web faz mais do que simplesmente mostrar a você uma informação estática, você pode apostar que tem JS envolvido. Ela abrange desde a rolagem que acontece quando você clica no menu da página, até animações audiovisuais em 2D e 3D.
