Seletores Combinacionais

Todas combinações podem ser feitas com qualquer seletor básico, aqui darei apenas alguns exemplos de como elas são feitas

  • Descendente: Ex: div a

    Seleciona todos elementos <a> descendentes de <div>

div a {
			 font-family: Arial;
			 color: black;
}
  • Filho direto: Ex: div > a

    Seleciona apenas elementos que são descendentes direto do primeiro

div > a {
				 color: red;
				 font-size: 14pt;
}
  • Irmão geral: Ex: div ~ a

    Seleciona os que são irmãos do primeiro e vem depois dele

div ~ a {
				font-family: Arial;
				color: black;
}

  • Irmão adjacente: Ex: div + a

    Seleciona os elementos que são irmãos do primeiro e vem diretamente após o ele

div + a {
		color: red;
		font-size: 14pt;
}
  • Ou: Ex: div, a

    Seleciona os elementos que correspondem a qualquer seletor na lista 

div, a {
		color: red;
		font-size: 14pt;
}
  • E: Ex: div.c

    Seleciona os elementos correspondentes à cobinação (no caso todas <div> que possuem a class "c") 

div.c {
		color: red;
		font-size: 14pt;
}
