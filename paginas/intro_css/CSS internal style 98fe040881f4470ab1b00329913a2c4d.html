CSS internal style

Para aplicar estilos de forma mais dinâmica e prática, podemos adicionar uma tag <style> dentro da área <head> do nosso documento HTML

Ainda deixando as configurações que fizemos anteriormente, adicione o código dentro da <head>

<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>CSS internal style</title>
    <style>
        body{
            font-family: Arial, Helvetica, sans-serif;
            background-color: lightgray;
            color: blue;
        }
        h1{
            color: green;
        }
    </style>
</head>

Vamos começar a usar classes nas tags HTML

Aqui vai um exemplo, criei a classe "azul" e a classe "vermelho" as quais vc deve configurar para deixar a letra azul com fundo azul claro, e a letra vermelha, respectivamente:

<h1 class="azul">Capítulo 1</h1>
<h2 class="vermelho">Capítulo 1.1</h2>
<p>
  Lorem ipsum dolor sit, amet consectetur adipisicing elit. Nostrum voluptatem 
</p>
<h2 class="vermelho">Capítulo 1.2</h2>
<p>
        Lorem ipsum dolor sit amet, consectetur adipisicing elit. Itaque, dolorem
</p>
<h1 class="azul">Capítulo 2</h1>
<h2>Capítulo 2.1</h2>
<p class="vermelho">
   Lorem ipsum dolor sit, amet consectetur adipisicing elit.
</p>
