Principais Formatações HTML

Negrito

Para escrever em negrito, utilizamos a tag <strong></strong>

Essa tag indica que os termos delimitados possuem força dentro da frase, e por tanto, aparecerão em negrito

<p>
	Nessa frase temos um <strong> termo em destaque </strong> usando strong
</p>

Itálico

A tag <em> significa que queremos dar ênfase (do Inglês emphasis) ao termo, logo ele ficará em itálico

<p>
   Nessa frase temos um <em> termo com enfase </em> usando em
</p>

Marca texto

Em HTML5 podemos marcar o texto como se estivéssemos usando uma caneta marca texto.

A tag <mark></mark> serve para marcar uma parte do texto na qual você quer dar uma ênfase descomunal, já que se trata de um trecho muito valioso para você

<p>
   Nessa frase temos um <mark> termo com enfase descomunal </mark> usando mark
</p>

Menor

Existe também a tag <small></small> que deixa os textos menores

<p>
   Nessa frase temos um <small> termo menor </small> usando small
</p>

Deletado

Temos a tag <del></del>, que significa que o texto que está ali pode até ser lido, mas deve ser desconsiderado 

<p>
   Nessa frase temos um <del> termo desconsiderado </del> usando del
</p>

Sublinhado

Se colocarmos um texto qualquer dentro de <ins></ins> estamos dizendo que o texto está ali, deve ser lido e você deve prestar atenção nele.

<p>
   Nessa frase temos um <ins> termo sublinhado </ins> usando ins
</p>

Sobrescrito e Subscrito

HTML tem as tags <sub></sub> e <sup></sup> para escrevermos por exemplo, a fórmula da água, ou expressões matemáticas 

<p>
    x<sup>2</sup>  H<sub>2</sub>O
</p>

Citações

Para criar uma citação em HTML, podemos usar a tag <q></q>. O texto já vai receber automaticamente as aspas, mas não terá nenhum deslocamento. Essa técnica é mais usada quando queremos uma citação no meio de um parágrafo.

Também podemos criar citações mais longas (em bloco) e que tenham um parágrafo só para si. Nesse caso, colocaremos tudo dentro de <blockquote> </blockquote> e o texto ganha um recuo automaticamente.

Podemos também colocar um link para o texto original, usando o parâmetro cite dentro da tag.

<q>
    Soluções em tecnologia. Para todos os tipos de negócios.
</q>
<blockquote cite="ijunior.com.br">
        Soluções em tecnologia.
        Para todos os tipos de negócios.
</blockquote>
